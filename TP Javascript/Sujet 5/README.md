## Pour le lancer localement

Cloner le projet

```bash
  git clone https://github.com/MaximeNGY/TP/new/main/TP%20Javascript/Sujet%205
```

Aller au répertoire projet

```bash
  cd Sujet%205
```

Installer les dépendances

```bash
  npm install express
```

Lancer le serveur

```bash
  node app.js
```
