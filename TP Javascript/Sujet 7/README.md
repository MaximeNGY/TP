
# Sujet 7 : Framework Vue.js




## Pour le lancer localement

Cloner le projet

```bash
  git clone https://github.com/MaximeNGY/TP/tree/main/TP%20Javascript/Sujet%207/vue-hello
```

Aller au répertoire projet

```bash
  cd vue-hello
```

Installer les dépendances

```bash
  npm install
```

Lancer le serveur

```bash
  npm run dev
```
